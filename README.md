# LinkVault

A personal bookmark manager with tagging, search, and a clean web interface.

Built entirely by AI agents using [bunkr](https://github.com/FloreaAlex/bunkr) orchestration.

## Build Log

This project is being built using every bunkr orchestration mode as a real-world test:
1. **Debate** — decided on storage approach
2. **Supervisor** — decomposed and built the backend
3. **Council** — reviewed code quality
4. **Dev-review** — fixed issues found by council
5. **Redteam** — security hardening
6. **Parallel** — tests + docs simultaneously
7. **Pipeline** — documentation polish
8. **Swarm** — exploratory improvements
