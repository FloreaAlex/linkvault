# LinkVault

> **This repository is built and maintained entirely by AI agents.** No human wrote any code in this project. Every line was produced by agents orchestrated through [bunkr](https://github.com/FloreaAlex/bunkr) — a multi-agent orchestration platform.

A personal bookmark manager with tagging, search, and a clean web interface.

## How This Was Built

Each phase used a different bunkr orchestration mode, with agents making all technical decisions:

| Phase | Mode | What Happened |
|-------|------|--------------|
| 1 | **Debate** | Two specialists argued storage approach; judge ruled |
| 2 | **Supervisor** | Manager decomposed the project; workers built it |
| 3 | **Council** | 3 reviewers voted on code quality |
| 4 | **Dev-review** | Developer fixed issues; reviewer approved |
| 5 | **Redteam** | Red team attacked; blue team hardened |
| 6 | **Parallel** | Tests + docs written simultaneously |
| 7 | **Pipeline** | Research → write → review documentation |
| 8 | **Swarm** | Agents explored and improved freely |

**Model:** GPT-OSS-120B (running locally on DGX Spark)

## Agent Instructions

Agents received product-manager-level instructions only. No technology was prescribed — they chose the stack, architecture, and implementation approach using their expertise and tools.
